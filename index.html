HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The html opening tag represents the root, or start, of the HTML document.
  • The title element defines the title of the webpage, which will appear on the title bar of the browser.
  • The base element defines the base URL for a webpage.
  • The link element connects exterbnal resource to the HTML document.
  • The meta element defines metadata such as character set, description, keywords, author and viewport.
  • The script element embeds client-side scripts in the HTML document.

CSS

  • There are three ways of inserting a style sheet: External CSS, Internal CSS and Inline CSS.
  • All HTML elements can be considered as having a box wrapped around them. This box is known as the Box Model.
  • The box model is made up of Content, Padding, Border and Margin.
  • Margin indicates how much space we want around the outside of an element. It creates space between the different elements (boxes) on the page.
  • Padding indicates how much space we want around the content inside an element.
  • In CSS, selectors are patterns used to select the element(s) to be styled.
  • A class selector selects all elements with that class name.
  • An element selector selects all the elements of that tag.
  • The wild card (*) selector selects all the elements in the HTML.

Git

  • git status: checks what branch we are currenlty on and the status of our git workflow.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add.: add all pending changes to the staging area.
  • git commit -m "Enter your commit message here": captures a snapshot of the staged changes.
  • git push: uploads the local repository toa remote repository.
  • git pull origin main: used to sync the local branch with the remote branch in GitHub.

JavaScript

  • Control flow is the order in which a computer executes code in a script.
  • The computer runs code from top to bottom in JavaScript.
  • A variable is a named container that allows us to store data in our code.
  • An array is a special variable, which can hold multiple items (values) under a single variable name.
  • Loops are used to execute the same code a number of times, each time with a different value.
  • The for loop repeats until a specified condition evaluates to false.
  • A function is a block of code designed to perform a particular task when "something" invokes it (calls it).
  • A function must be defined somewhere in the scope from which you wish to call it
